# java-calculator-precourse

## 프로그램 설명
입력한 문자열에서 숫자를 추출하여 더하는 계산기를 구현한다.

## 기능 구현 목록

### 입출력
- [x] 사용자로부터 계산할 문자열을 입력받는다.
- [ ] 계산 결과를 출력한다.

### 구분자
- [x] 커스텀 구분자가 존재하는지 확인한다.
  - [ ] 존재한다면, 분리해서 저장한다.

### 파싱
- [ ] 기본 구분자 ',', ':'와 커스텀 구분자를 이용하여 피연산자를 구한다.

### 계산기
- [ ] 피연산자를 기반으로 덧셈을 수행한다.

### 예외 상황
- [ ] 숫자 범위에 대한 예외
- [x] 커스텀 구분자가 문자가 아닌 경우
- [ ] 피연산자가 양수가 아닌 경우
- [ ] 입력 형식이 올바르지 않은 경우
- [ ] 커스텀 구분자가 기본 구분자에 포함되는 경우
